## 프로젝트
충북대학교 소프트웨어학과 과목별 게시판

![image](https://user-images.githubusercontent.com/81895590/205443664-2e0728e1-b6a8-4cf1-bd40-4455f1c71ac2.png)

## 기능

1. 로그인 / 회원가입
2. 글 작성 및 수정 삭제 / 글 카테고리 분류
3. 과목 별 게시글 분류
4. 해당 과목 수업시간 및 강의실 정보 띄우기
5. 댓글 
6. 팀 매칭 기능
